Sushi

Type lorem({word_count}, {paragraph_count}) in your file, make sure that the input curosr is within the region of the lorem{?, ?} expression, then press the shortcut key (alt+l). Parameter "word_count" represents how many paragraphs to insert and parameter "paragraph_count" represents how many words that each paragraph contains. For example, lorem(100, 3) means that it will create three paragraphs of lorem ipsum text and each paragraph contains one hundred words. Both the two parameters are optional, if you leave them empty, the plugin will use default values that are configured in the settings file.
